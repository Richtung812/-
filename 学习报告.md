## <a id='3'>学习内容3</a>
**Git工作区域**   
工作区    暂存区     Git仓库

**Git 基础设置**
* 设置用户名      
`git config -global user.name"你的用户名"`
* 设置邮箱     
`git config -global user.email"你的邮箱"`
* 查看设置     
`git config --list`

**Git 仓库的创建和操作**   
* 创建文件夹     
`mkdir demo1`
* 创建Git仓库(在文件夹内初始化Git)     
`cd demo1`    
`git init`    
* 创建文件   
`touch a1.php`       
* 将文件提交到暂存区    
`git add a1.php`  
* 提交操作    
`git commit -m"提交描述"` 
* 删除文件    
`rm a1.php`
* 从Git中删除   
`git rm a1.php`
